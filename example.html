CNJS Carousel

The included CSS stylesheet is for a responsive carousel and defaults to using percentages based on 10 items. The JS will re-calculate the percentages based on actual length of items.

When setting the 'isResponsive' option to false, the JS will not calculate relative sizes, and you must ensure the CSS units are accurate.

Main Carousel

"Hero" style main carousel uses class defaults: {numVisibleItems: 1, numItemsToAnimate: 1, isResponsive: true, enableSwipe: true}, and custom options: {highlightActive: true, autoRotate: true}
Uses CSS opacity transition to fade in/out the media-info area when active / highlighted class is applied.

Thumb Carousel

The thumb carousel uses custom options: {numVisibleItems: 5, numItemsToAnimate: 2}
The numVisibleItems option must be accurate or weird stuff will happen. The numItemsToAnimate can be set as desired (within reason of course).
